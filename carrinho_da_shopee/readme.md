🛒 Shopee Cart Simulator (Node.js)
📌 Descrição

Este projeto é um simulador de carrinho de compras inspirado na Shopee, desenvolvido com Node.js e executado no terminal.

O sistema permite que o usuário:

Adicione produtos ao carrinho
Remova produtos
Atualize quantidades
Visualize o carrinho completo
Calcule automaticamente o total da compra

Toda a lógica de gerenciamento do carrinho é controlada no backend, simulando a experiência de um e-commerce real.

🚀 Funcionalidades
✅ Adicionar produtos
✅ Remover produtos
✅ Atualizar quantidade de itens
✅ Listar produtos no carrinho
✅ Calcular total automaticamente
✅ Persistência de dados em arquivo JSON
✅ Menu interativo no terminal
🧠 Tecnologias utilizadas
Node.js
JavaScript (ES Modules)
Readline para interação no terminal
File System (fs) para persistência
