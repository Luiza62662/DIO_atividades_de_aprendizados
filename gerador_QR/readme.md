# 🧾 Gerador de QR Code para E-commerce

## 📌 Descrição

Este projeto consiste em um gerador de QR Codes via terminal desenvolvido com Node.js.  
A aplicação permite gerar QR Codes a partir de links de produtos, facilitando o acesso direto a páginas de vendas online.

---

## 🎯 Objetivo

Criar uma ferramenta simples e eficiente para geração de QR Codes sem necessidade de interface gráfica.

---

## ⚙️ Tecnologias

- Node.js
- JavaScript
- Biblioteca qrcode
- prompt-sync

---

## 🚀 Funcionalidades

- Geração de QR Code via terminal
- Entrada dinâmica de links
- Exportação em PNG
- Código modularizado
